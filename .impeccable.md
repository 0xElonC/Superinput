# Design Context

## Target audience
People who type across languages in desktop chat, mail, browser, and work tools. They need a compact assistant that stays out of the way while they write.

## Use cases
- Preview a target-language translation while composing text.
- Quickly test and configure an OpenAI-compatible agent endpoint.
- Clear the translation after Enter so the next sentence starts fresh.

## Brand personality/tone
Quiet, utilitarian, precise, and privacy-aware. The interface should feel like a professional desktop utility, not a marketing product or chatbot landing page.
